## Fast API todo list

### How to use
```shell
docker-compose up --build
```
Test with API client services such as Postman, Insomnia, Paw, etc.

### Features
- User signup / login
- CRUD the user's own tasks.

### Used libraries
FastAPI, SQLAlchemy, Pydantic
